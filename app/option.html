소개

  • 웹페이지에 이미지를 손쉽게 띄우고, 이동시킬 수 있습니다.
  • We do not yet support English. But it will not be difficult to use. Use it as you think.

사용법 안내

기본 기능
  • 1. 이미지 파일선택
  • 2. 방향키를 이용하여 위치 조절

추가 기능
  • 툴바에서 선택
  • 1. 이미지 숨기기/보이기
  • 2. 이미지 비율 조절
  • 2. 이미지 투명도 조절

기본키
  • arrow key {up, down, left, right} : 1px move
  • shift + arrow key : 10px move
  • 이동이 안되면 이미지 클릭 후, 조작

단축키
  • 실행/해제 단축키를 설정하면 빠르게 전환할 수 있습니다.
  • 1. chrome://extensions/로 이동
  • 2. 확장프로그램 목록 최하단에 "단축키" 링크 선택
  • 3. "프로그램명 - 확장 프로그램 활성화" 항목에서 단축키 설정
